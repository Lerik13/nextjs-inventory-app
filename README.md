## Inventory Management App

A full-stack Inventory Management application with user authentication, dashboard analytics, product management, and real-time inventory tracking. Built to help businesses monitor stock levels, manage products efficiently, and gain insights through visual analytics.

### Features

- ##### User Authentication

  Secure sign-up and login

- ##### Dashboard Analytics

  Overview of total products and stock levels
  Low-stock alerts
  Stock levels
  Chart with new products per week
  Efficiency

- ##### Inventory or Product Management

  Create, and delete products
  Search product by name and SKU
  Track price, quantity, SKU, and low-stock thresholds

- ##### Real-Time Inventory Tracking

  Instant updates on inventory changes
  Accurate stock visibility at all times

- ##### User-Friendly Interface
  Clean, responsive UI
  Optimized for desktop and mobile

### Tech Stack

- Next.js 15 – React framework with App Router and Server Components
- React 19 – Component-based UI development with latest features
- TailwindCSS – Utility-first CSS for modern styling
- Stack Auth – Modern authentication solution (replaces NextAuth.js)
- Prisma – Type-safe database ORM with migrations
- PostgreSQL – Robust relational database
- Lucide Icons – Clean and beautiful icon pack
- Recharts – Data visualization for analytics
- TypeScript – Type safety and enhanced developer experience
